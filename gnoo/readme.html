Gnoo Skin

Le Gnoo skin pour Nucleus, une réalisation de Globule.


Ce skin utilise des includes pour les différentes parties du skin :

head.inc - html de l'entete

header.inc - entete de la page

sidebar.inc - menu de la page

footer.inc - bas de page


Note: pensez à passer les fichiers .inc du dossier /skin/gnoo en chmod 666.

Le skin contient un dossier "blank_images" (images vierges) dans lequel vous trouverez un bouton ainsi que la police utilisée
